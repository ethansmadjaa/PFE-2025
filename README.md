## Prérequis

- Python 3.x
- UV (gestionnaire de paquets Python moderne)

## Installation

1. **Installer UV**

   ```bash
   pip install uv
   ```

2. **Synchroniser les dépendances**

   ```bash
   uv sync
   ```

## Lancement

### Démarrer l'application principale

```bash
uv run main.py
```

### Exécuter les tests

Dans un terminal séparé, lancez :

```bash
uv run test.py
```
